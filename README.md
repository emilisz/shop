## Simple responsive shop example

Made with laravel, tailwindcss and livewire.



## How to

- Download
- run ``` composer install ```
- run ``` npm install ```
- run ``` php artisan key:generate ```
- run ``` php artisan migrate --seed ```
- run ``` php artisan serve ```
